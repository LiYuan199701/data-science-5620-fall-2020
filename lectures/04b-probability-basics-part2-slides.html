BACK TO AND

If events are independent, \[P(Z = z\ \text{ and }\ W = w) = P(Z = z)\ P(W = w)\]

FROM AND TO GIVEN

More generally, \[P(Z = z\ \text{ and }\ W = w) = P(Z = z)\ P(W = w|Z = z)\]

FROM GIVEN TO AND

\[P(W = w|Z = z) = \frac{P(Z = z\ \text{ and }\ W = w)}{P(Z = z)}\]

Remember: Take advantage of disjoint to break into small parts

\[P(Z = z) = \sum_{j = \{\text{all values of W}\}} P(Z = z\ \text{ and }\ W = j)\]

Now break up event into smaller parts, change and to given

\[P(Z = z) = \sum_{j = \{\text{all values of W}\}} P(W = j)P(Z = z | W = j)\]

FROM (W given Z) to (Z given W) (Bayes Rule)

\[P(W = w|Z = z) = \frac{P(Z = z\ \text{ and }\ W = w)}{P(Z = z)}\]

\[P(W = w|Z = z) = \frac{P(W = w)P(Z = z|W = w)}{\sum_{\text{all } j} P(W = j) P(Z = z | W = j)}\]
